# DinoGame 

A fun, interactive web-based clone of the classic Chrome T-Rex infinite runner game. Built using pure frontend technologies, players can control a pixelated Tyrannosaurus rex running across a side-scrolling landscape, jumping over obstacles to achieve the highest score possible, made by Arpan Poudel.

 # Live Demo

Check out the live deployment here: [dinogamebyarpan.netlify.app](https://dinogamebyarpan.netlify.app)

---

# Features

  Classic Gameplay: Smooth infinite runner mechanics mimicking the original Chrome easter egg.
  Score Tracking: Keeps track of your current score.
  Responsive Design: Playable across different screen sizes.




# Built With

HTML5: For the structural layout and game canvas.
CSS3: For styling, pixel-art presentation, and layout responsiveness.
JavaScript: For game physics, collision detection, movement controls, and the infinite loop logic.
